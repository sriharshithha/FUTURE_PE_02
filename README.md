

FUTURE_PE_02  
AI-powered UGC ad script generation system for a conceptual local café brand “Velvet Café” built using structured prompt engineering.

---

**Project Overview**

This project presents an AI-generated UGC script created for Velvet Café, targeting college students in Vizianagaram. The objective was to generate authentic, relatable, and conversion-focused promotional content using structured prompt engineering.

**UGC Script**

 **Hook**
  - “I just wanted coffee… but this place surprised me.”
  - “I think I just found the best study spot in town.”

 **Problem**
  - Every café I visited was either too crowded or too expensive.
  - Finding a calm and affordable place to study felt almost impossible.

  **Solution**
  - I discovered Velvet Café in Vizianagaram.
  - It is student-friendly and offers a feature called Solo Space — a private, quiet area designed for focused study and relaxation.

 **Benefits**
  - Comfortable and calm study environment  
  - Affordable pricing for students  
  - Freshly prepared food with live pizza preparation  

  **Call to Action**
  - If you're a college student in Vizianagaram, this place is worth visiting.

**Objective**

• To design a structured AI prompt capable of generating clear and persuasive UGC scripts.  

**Methodology**

• Used a predefined content framework to guide AI output.  
• Ensured emotional relatability and marketing clarity.  

**Tools Used**

• ChatGPT – Content generation  
• GitHub – Documentation and version control 



**DOCUMENTATION OF "VELVET CAFE CAFE" WEBSITE**

VELVET CAFE – AI POWERED BUSINESS WEBSITE  
A premium, responsive café website designed using structured AI-assisted development.

------------------------------------------------------------

 **PROJECT OVERVIEW**

This project represents the development of a modern, responsive business website for a conceptual café brand. The website is designed to showcase menu offerings, enable booking interaction, and create a premium digital presence suitable for student audiences.

The goal was to combine AI-assisted content generation with frontend implementation to build a visually appealing and functional website.

------------------------------------------------------------

 **OBJECTIVES**

• Develop a clean, premium café website  
• Implement Indian currency localization (₹)  
• Integrate interactive features (smooth scroll, booking action)  
• Ensure responsive design for mobile and desktop  
• Demonstrate AI-assisted development workflow  

------------------------------------------------------------

**WEBSITE FEATURES**

• Hero section with immersive café background  
• Smooth scrolling navigation  
• Menu section with product cards  
• Indian Rupee pricing (₹)  
• Booking form with interactive submission  
• Hover animations and transition effects  
• Mobile responsive layout  
• Dynamic image integration using external sources  

------------------------------------------------------------

USER INTERFACE DESIGN

The website follows a dark luxury theme with warm highlights to create a premium café ambiance.

Design considerations include:

• Clean typography for readability  
• Balanced spacing and section alignment  
• Card-based layout for menu presentation  
• Soft hover animations for better user engagement  
• Clear call-to-action buttons  

The layout ensures visual hierarchy and structured content flow.

------------------------------------------------------------

TECHNICAL IMPLEMENTATION

Frontend Technologies Used:

• HTML5 – Structure  
• CSS3 – Styling and animations  
• JavaScript – Interactivity  
• Unsplash dynamic image source – Visual enhancement  

 **Key Implementations**
